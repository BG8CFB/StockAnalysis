## 账号申请与配置

### 申请 Token

1. 访问 [iTick 官网](https://www.itick.io)
2. 注册账号
3. 登录后获取 API Token
4. Token 用于所有 API 请求的身份验证

**注意**：
- 免费 Token 有访问限制
- 付费 Token 提供更高限额和更多功能
- 请妥善保管 Token

### Token 配置

配置方式：
1. **环境变量**：`ITICK_API_TOKEN`
2. **配置文件**：存储在项目配置中
3. **加密存储**：推荐在生产环境使用加密存储

### 请求头设置

所有 API 请求需要包含以下请求头：

```
token: Your_Token_Here
accept: application/json
```

## 美股接口

### 1. 股票信息

**功能**：获取美股股票基本信息

**接口地址**：`/stock/info`

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `type` | string | ✅ | 资产类型，固定值 `stock` |
| `region` | enum | ✅ | 市场代码：US（美国） |
| `code` | string | ✅ | 产品代码（如 AAPL） |

**返回字段**：
- `c` - 股票代码
- `n` - 股票名称
- `t` - 资产类型
- `e` - 交易所
- `s` - 行业板块
- `i` - 行业细分
- `mcb` - 总市值
- `tso` - 总股本
- `pet` - 市盈率
- `fcc` - 货币代码

**示例请求**：
```
GET /stock/info?type=stock&region=US&code=AAPL
```

## 常见问题

### Token 问题

**问题：Token 无效**

原因：Token 错误或已过期

解决方案：
1. 检查 Token 是否正确
2. 登录 iTick 后台重新获取 Token
3. 检查 Token 是否过期

---

### 数据问题

**问题：返回数据为空**

可能原因：
1. 股票代码错误
2. 市场代码错误
3. Token 权限不足

解决方案：
1. 验证股票代码格式
2. 检查 region 参数是否正确（US）
3. 确认 Token 权限

---

**问题：数据延迟**

原因：免费版可能有延迟

解决方案：
1. 升级到付费版获取实时数据
2. 检查网络连接

---

### 接口调用问题

**问题：频率限制**

原因：超过免费版调用限制

解决方案：
1. 控制请求频率
2. 升级到付费版
3. 使用批量接口减少请求数

---

**问题：并发限制**

原因：超过并发连接限制

解决方案：
1. 减少并发连接数
2. 使用 WebSocket 订阅代替轮询
