## 美股接口

### 1. 实时行情数据

#### 1.1 美股实时行情

**接口名称**：`stock_us_spot()`

**功能描述**：获取美股所有上市公司的实时行情数据

**数据源**：新浪财经

**返回字段**：

| 字段名 | 说明 | 字段名 | 说明 |
|--------|------|--------|------|
| 序号 | 序号 | 代码 | 股票代码 |
| 名称 | 股票名称 | 最新价 | 最新成交价 |
| 涨跌额 | 涨跌金额 | 涨跌幅 | 涨跌幅百分比 |
| 开盘 | 开盘价 | 昨收 | 昨收价 |
| 最高 | 最高价 | 最低 | 最低价 |
| 成交量 | 成交量 | 成交额 | 成交额 |
| 市盈率 | 市盈率 | 市净率 | 市净率 |

**示例代码**：

```python
import akshare as ak

# 获取所有美股实时行情
df = ak.stock_us_spot()

# 筛选特定股票（如苹果）
stock_info = df[df['代码'] == 'AAPL']
print(stock_info)
```

### 2. 历史行情数据

#### 2.1 美股历史日线数据

**接口名称**：`stock_us_daily()`

> **测试状态**：✅ 可用 (需注意参数限制)
> **说明**：接口不支持 `start_date`/`end_date` 参数，调用时将返回**全部历史数据**。
> **建议**：获取数据后需在本地进行日期过滤。

**功能描述**：获取美股个股历史日线行情数据（全部历史）

**数据源**：新浪财经

**⚠️ 注意事项**：
- 接口不支持时间范围筛选，始终返回全部历史数据。
- 数据量较大，建议本地缓存。

**请求参数**：

| 参数名 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| symbol | str | 股票代码（如 "AAPL"） | - |
| adjust | str | 复权类型："" 不复权, "qfq" 前复权 | "" |

**返回字段**：

| 字段名 | 说明 |
|--------|------|
| date | 交易日期 |
| open | 开盘价（美元） |
| high | 最高价（美元） |
| low | 最低价（美元） |
| close | 收盘价（美元） |
| volume | 成交量（股） |

**示例代码**：

```python
import akshare as ak
import pandas as pd

# 获取苹果公司全部历史日线数据
df = ak.stock_us_daily(symbol="AAPL", adjust="qfq")

# 本地过滤日期（示例）
df['date'] = pd.to_datetime(df['date'])
df = df[(df['date'] >= '2024-01-01') & (df['date'] <= '2024-12-31')]
print(df.head())
```

## 限流与注意事项

### 1. 请求频率限制

AKShare 本身不强制限流，但依赖的第三方数据源有反爬机制。

**建议策略**：
- 单个接口请求间隔：0.5-1 秒
- 批量请求时添加随机延时
- 避免在短时间内重复请求相同数据

### 2. 并发控制

**推荐并发数**：
- 同一域名：不超过 3 个并发
- 不同数据源：可适当提高

### 3. 错误重试机制

**建议配置**：
- 最大重试次数：3 次
- 重试间隔：指数退避（1s, 2s, 4s）

### 4. 美股注意事项

**美股交易时间**：
- 常规交易时间：周一至周五 22:30-05:00（北京时间）
- 盘前交易：21:00-22:30
- 盘后交易：05:00-06:00

**美股代码格式**：
- 纽交所 (NYSE)：如 "AAPL", "JPM"
- 纳斯达克 (NASDAQ)：如 "GOOGL", "MSFT"
- 美国存托凭证 (ADR)：如 "BABA" (阿里巴巴)

## 常见问题与解决方案

### 1. ReadTimeout 错误

**解决方案**：
1. 重新运行接口函数
2. 降低访问频率
3. 增加超时时间

### 2. HTTP 403 错误

**原因**：请求过于频繁，触发反爬机制

**解决方案**：
1. 降低请求频率
2. 添加请求延时

### 3. 数据为空

**可能原因**：
1. 股票代码错误
2. 该股票已退市
3. 非交易时间

**解决方案**：
1. 验证股票代码格式
2. 检查股票是否正常交易
3. 确认是否在交易时间内
