## 1. 数据源简介

### 1.1 TuShare Pro 港股概述

**TuShare Pro** 提供完整的港股市场数据服务。

**港股数据特点**：
- ✅ **数据覆盖**: 覆盖港股主板、创业板
- ✅ **实时行情**: 支持日线和分钟级行情
- ✅ **财务数据**: 三大报表、财务指标
- ✅ **交易日历**: 港股交易日历
- ✅ **复权支持**: 支持复权数据（通过 pro_bar）

### 1.2 港股数据覆盖范围

| 数据类别 | 覆盖内容 |
|---------|---------|
| **股票数据** | 港股主板、创业板 |
| **行情数据** | 日线、分钟线、复权数据 |
| **财务数据** | 三大报表、财务指标 |
| **基础数据** | 股票列表、交易日历 |

### 1.3 数据更新频率

| 数据类型 | 更新频率 | 更新时间 |
|---------|---------|---------|
| **日线行情** | 每交易日 | 每日 18:00 左右 |
| **分钟行情** | 实时 | 交易时间内 |
| **财务数据** | 季度 | 财报披露后更新 |

## 2. 接入准备

### 2.1 注册账号

1. 访问 TuShare Pro 官网: https://tushare.pro
2. 点击右上角「注册」按钮
3. 填写用户名、邮箱、密码完成注册

### 2.2 获取 API Token

**获取步骤**：
1. 登录 TuShare Pro
2. 点击右上角「个人主页」
3. 点击左侧菜单「接口TOKEN」
4. 复制您的 Token

### 2.3 安装 Python SDK

```bash
pip install tushare
```

**版本要求**: >= 1.2.10

## 3. 调用方式

### 3.1 Python SDK（推荐）

```python
import tushare as ts

# 设置 Token
ts.set_token('your_token_here')

# 初始化 pro 接口
pro = ts.pro_api()

# 调用接口
df = pro.hk_basic(
    list_status='L',
    fields='ts_code,symbol,name,market,list_date'
)
```

### 3.2 通用行情接口（支持复权）

> **测试状态**: ✅ 港股复权已测试可用

```python
# 港股前复权日线数据
df = ts.pro_bar(
    ts_code='00700.HK',  # 腾讯控股
    adj='qfq',  # 前复权
    start_date='20230101',
    end_date='20231231'
)

# 港股后复权日线数据
df = ts.pro_bar(
    ts_code='00700.HK',
    adj='hfq',  # 后复权
    start_date='20230101',
    end_date='20231231'
)
```

## 5. 接口分类与详情

### 5.1 基础数据接口

#### 5.1.1 港股基础信息 - hk_basic

> **测试状态**: ✅ 已测试可用

**接口描述**: 获取港股基础信息列表

**请求参数**:

| 参数名 | 类型 | 必选 | 说明 |
|-------|------|------|------|
| list_status | str | N | 上市状态：L上市 D退市 P暂停上市 |
| ts_code | str | N | 股票代码 |

**返回字段**:

| 字段名 | 说明 | 示例 |
|-------|------|------|
| ts_code | 股票代码 | 00700.HK |
| symbol | 股票代码（不含后缀） | 00700 |
| name | 股票名称 | 腾讯控股 |
| fullname | 股票全称 | 腾讯控股有限公司 |
| enname | 英文名称 | Tencent Holdings Limited |
| market | 市场类型 | 主板 |
| exchange | 交易所代码 | HKEX |
| curr | 交易货币 | HKD |
| list_status | 上市状态 | L上市 |
| list_date | 上市日期 | 20040616 |
| is_hs | 是否沪深港通标的 | S(沪港通) H(深港通) N(非) |

**权限要求**: 2000 积分

**调用示例**:

```python
# 获取所有港股列表
df = pro.hk_basic(
    list_status='L',
    fields='ts_code,symbol,name,market,exchange,list_date,is_hs'
)

# 获取单个港股信息
df = pro.hk_basic(ts_code='00700.HK')
```

#### 5.1.2 港股交易日历 - hk_tradecal

> **测试状态**: ✅ 已测试可用（需要修正接口名为 hk_tradecal）

**接口描述**: 获取港股交易日历

**请求参数**:

| 参数名 | 类型 | 必选 | 说明 |
|-------|------|------|------|
| exchange | str | N | 交易所：HKEX，默认HKEX |
| start_date | str | N | 开始日期，格式：YYYYMMDD |
| end_date | str | N | 结束日期，格式：YYYYMMDD |
| is_open | str | N | 是否开盘：0休市 1交易 |

**返回字段**:

| 字段名 | 说明 |
|-------|------|
| exchange | 交易所代码 |
| cal_date | 日历日期 |
| is_open | 是否交易（1是 0否） |
| pretrade_date | 上一交易日 |

**权限要求**: 2000 积分

**调用示例**:

```python
# 获取 2023 年港股交易日历
df = pro.hk_trade_cal(
    exchange='HKEX',
    start_date='20230101',
    end_date='20231231',
    is_open='1'
)
```

### 5.2 行情数据接口

#### 5.2.1 港股日线行情 - hk_daily

> **测试状态**: ⚠️ 测试失败 - 权限不足（当前积分每天最多访问 10 次，需要更高积分）

**接口描述**: 获取港股每日增量和历史行情数据

**请求参数**:

| 参数名 | 类型 | 必选 | 说明 |
|-------|------|------|------|
| ts_code | str | N | 股票代码（支持多个，用逗号分隔） |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| fields | str | N | 指定返回字段 |

**返回字段**:

| 字段名 | 说明 | 数据类型 |
|-------|------|---------|
| ts_code | 股票代码 | str |
| trade_date | 交易日期 | str |
| open | 开盘价 | float |
| high | 最高价 | float |
| low | 最低价 | float |
| close | 收盘价 | float |
| pre_close | 昨收价 | float |
| change | 涨跌额 | float |
| pct_chg | 涨跌幅(%) | float |
| vol | 成交量（手） | float |
| amount | 成交额（千元） | float |

**权限要求**: 2000 积分
**更新时间**: 每日 18:00 左右
**限量**: 单次最大 5000 行记录

**调用示例**:

```python
# 获取腾讯控股日线数据
df = pro.hk_daily(
    ts_code='00700.HK',
    start_date='20230101',
    end_date='20231231'
)

# 获取所有港股某一天的数据
df = pro.hk_daily(trade_date='20230103')
```

**注意**: 基础积分（2000分）下，此接口每天最多调用 10 次。

#### 5.2.2 港股复权行情 - hk_daily_adj

> **测试状态**: ✅ 港股复权专用接口

**接口描述**: 获取港股复权行情数据（专用接口，替换 pro_bar）

**支持市场**: 港股
**复权支持**: 前复权、后复权

**请求参数**:

| 参数名 | 类型 | 必选 | 说明 |
|-------|------|------|------|
| ts_code | str | N | 股票代码（支持多个，用逗号分隔） |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| adj_type | str | N | 复权类型：None不复权 qfq前复权 hfq后复权 |

**返回字段**:

| 字段名 | 说明 | 数据类型 |
|-------|------|---------|
| ts_code | 股票代码 | str |
| trade_date | 交易日期 | str |
| open | 开盘价 | float |
| high | 最高价 | float |
| low | 最低价 | float |
| close | 收盘价 | float |
| vol | 成交量（手） | float |
| amount | 成交额（千元） | float |
| adj_factor | 复权因子 | float |

**权限要求**: 120 积分（试用）
**更新时间**: 每日 18:00 左右
**限量**: 单次最大 6000 行记录

**调用示例**:

```python
# 获取腾讯控股前复权日线数据
df = pro.hk_daily_adj(
    ts_code='00700.HK',
    start_date='20230101',
    end_date='20231231',
    adj_type='qfq'
)

# 获取腾讯控股后复权日线数据
df = pro.hk_daily_adj(
    ts_code='00700.HK',
    start_date='20230101',
    end_date='20231231',
    adj_type='hfq'
)
```

**推荐**: 使用 `hk_daily_adj` 接口获取港股复权数据，无调用次数限制。

**注意**: `pro_bar` 接口不支持港股，请使用 `hk_daily_adj` 替代。

## 6. 数据字段说明

### 6.1 通用字段约定

**股票代码格式**: `{代码}.{交易所}`
- 港交所：如 `00700.HK`、`9988.HK`

**日期格式**: `YYYYMMDD`
- 示例：`20230101`、`20231231`

**金额单位**:
- 成交额：千元（日线数据）
- 财务数据：元、万元、亿元

**数量单位**:
- 成交量：手（1手=100股）

### 6.2 复权类型

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| None | 不复权 | 查看原始价格 |
| qfq | 前复权 | 技术分析（推荐） |
| hfq | 后复权 | 长期趋势分析 |

**港股复权说明**: 港股通过 `hk_daily_adj` 专用接口获取复权数据（`pro_bar` 不支持港股）。

## 7. 错误处理与最佳实践

### 7.1 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| hk_daily 调用次数限制 | 基础积分每天最多 10 次 | 使用 `hk_daily_adj` 替代 |
| pro_bar 不支持港股 | pro_bar 仅支持 A 股 | 使用 `hk_daily_adj` 专用接口 |
| 接口名错误 | 文档与实际不一致 | 使用 `hk_tradecal` 而非 `hk_trade_cal` |

### 7.2 最佳实践

1. **推荐使用 `hk_daily_adj` 接口**获取港股复权数据
   - 120 积分即可试用
   - 支持前复权/后复权
   - 无调用次数限制

2. **港股日线限制**
   - `hk_daily` 在基础积分下每天最多调用 10 次
   - 建议使用 `hk_daily_adj` 作为替代

3. **积分等级说明**
   - 120 积分：可试用 `hk_daily_adj`（港股复权行情）
   - 2,000 积分：可访问港股全部接口（基础信息、日线、财务数据）

4. **财务数据完整性**
   - 港股三大报表接口均正常可用
   - 财务指标数据完整
