## 1. 数据源简介

### 1.1 TuShare Pro A股概述

**TuShare Pro** 是中国领先的金融数据服务平台，A股数据是其核心优势。

**A股数据特点**：
- ✅ **数据权威**: 直接来源于上交所、深交所
- ✅ **数据完整**: 覆盖所有A股上市公司
- ✅ **更新及时**: 日线数据收盘后更新，分钟数据实时推送
- ✅ **历史悠久**: A股历史数据可追溯至1990年
- ✅ **数据丰富**: 基础行情、财务数据、宏观指标、龙虎榜等

### 1.2 A股数据覆盖范围

| 数据类别 | 覆盖内容 |
|---------|---------|
| **股票数据** | 沪深主板、科创板、创业板、北交所 |
| **指数数据** | 沪深指数、中证指数、行业指数、概念指数 |
| **行情数据** | 日线、分钟线、实时报价、复权数据 |
| **财务数据** | 三大报表、财务指标、业绩预告 |
| **公司数据** | 股东信息、高管信息、公司公告 |
| **特色数据** | 龙虎榜、资金流向、融资融券 |

### 1.3 数据更新频率

| 数据类型 | 更新频率 | 更新时间 |
|---------|---------|---------|
| **日线行情** | 每交易日 | 15:00-17:00 |
| **分钟行情** | 实时 | 交易时间内实时推送 |
| **财务数据** | 季度 | 季报披露后更新 |
| **公司公告** | 实时 | 公告发布后更新 |

## 2. 接入准备

### 2.1 注册账号

1. 访问 TuShare Pro 官网: https://tushare.pro
2. 点击右上角「注册」按钮
3. 填写用户名、邮箱、密码完成注册
4. 登录后进入个人主页

### 2.2 获取 API Token

**获取步骤**：
1. 登录 TuShare Pro
2. 点击右上角「个人主页」
3. 点击左侧菜单「接口TOKEN」
4. 复制您的 Token

### 2.3 安装 Python SDK

```bash
pip install tushare
```

**版本要求**: >= 1.2.10

## 3. 调用方式

### 3.1 Python SDK（推荐）

```python
import tushare as ts

# 设置 Token
ts.set_token('your_token_here')

# 初始化 pro 接口
pro = ts.pro_api()

# 调用接口
df = pro.stock_basic(
    exchange='',
    list_status='L',
    fields='ts_code,symbol,name,area,industry,list_date'
)
```

### 3.2 通用行情接口（支持复权）

```python
# A股前复权日线数据
df = ts.pro_bar(
    ts_code='600519.SH',
    adj='qfq',  # 前复权
    start_date='20230101',
    end_date='20231231'
)

# A股后复权日线数据
df = ts.pro_bar(
    ts_code='600519.SH',
    adj='hfq',  # 后复权
    start_date='20230101',
    end_date='20231231'
)

# A股不复权日线数据
df = ts.pro_bar(
    ts_code='600519.SH',
    adj=None,  # 不复权
    start_date='20230101',
    end_date='20231231'
)
```

## 5. 接口分类与详情

### 5.1 基础数据接口

#### 5.1.1 股票列表 - stock_basic

**接口描述**: 获取A股股票基础信息

**请求参数**:

| 参数名 | 类型 | 必选 | 说明 |
|-------|------|------|------|
| ts_code | str | N | 股票代码（如：600519.SH） |
| list_status | str | N | 上市状态：L上市 D退市 P暂停上市 |
| exchange | str | N | 交易所：SSE上交所 SZSE深交所 |
| fields | str | N | 指定返回字段 |

**返回字段**:

| 字段名 | 说明 | 示例 |
|-------|------|------|
| ts_code | 股票代码 | 600519.SH |
| symbol | 股票代码（不含后缀） | 600519 |
| name | 股票名称 | 贵州茅台 |
| area | 所在地域 | 贵州 |
| industry | 所属行业 | 酒、饮料和精制茶制造业 |
| market | 市场类型 | 主板 |
| exchange | 交易所代码 | SSE |
| list_date | 上市日期 | 20010827 |
| is_hs | 是否沪深港通标的 | S(沪股通) H(深股通) N(非) |

**权限要求**: 0 积分

**调用示例**:

```python
# 获取所有上市A股
df = pro.stock_basic(
    exchange='',
    list_status='L',
    fields='ts_code,symbol,name,area,industry,market,exchange,list_date'
)
```

#### 5.1.2 交易日历 - trade_cal

**接口描述**: 获取A股交易日历

**请求参数**:

| 参数名 | 类型 | 必选 | 说明 |
|-------|------|------|------|
| exchange | str | N | 交易所：SSE上交所 SZSE深交所 |
| start_date | str | N | 开始日期，格式：YYYYMMDD |
| end_date | str | N | 结束日期，格式：YYYYMMDD |
| is_open | str | N | 是否开盘：0休市 1交易 |

**返回字段**:

| 字段名 | 说明 |
|-------|------|
| exchange | 交易所代码 |
| cal_date | 日历日期 |
| is_open | 是否交易（1是 0否） |
| pretrade_date | 上一交易日 |

**权限要求**: 0 积分

**调用示例**:

```python
# 获取 2023 年上交所交易日历
df = pro.trade_cal(
    exchange='SSE',
    start_date='20230101',
    end_date='20231231',
    is_open='1'
)
```

**权限要求**: 0 积分

    ts_code='600519.SH',

    end_date='20231231'

df = ts.pro_bar(
    ts_code='600519.SH',
    freq='W',

## 6. 数据字段说明

### 6.1 通用字段约定

**股票代码格式**: `{代码}.{交易所}`
- SSE（上交所）：如 `600519.SH`、`000001.SH`
- SZSE（深交所）：如 `000001.SZ`、`300001.SZ`

**日期格式**: `YYYYMMDD`
- 示例：`20230101`、`20231231`

**金额单位**:
- 成交额：元（分钟数据）、千元（日线数据）
- 财务数据：元、万元、亿元

**数量单位**:
- 成交量：手（1手=100股）

### 6.2 常用复权类型

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| None | 不复权 | 查看原始价格 |
| qfq | 前复权 | 技术分析（推荐） |
| hfq | 后复权 | 长期趋势分析 |

## 7. 错误处理与最佳实践

### 7.1 常见错误码

| 错误码 | 错误信息 | 解决方案 |
|-------|---------|---------|
| -2000 | 用户积分不足 | 提升积分等级 |
| -2002 | 每分钟调取超过限制 | 添加延时，降低频次 |
| -2003 | 无权限调取该接口 | 检查积分是否满足要求 |

### 7.2 限流建议

| 积分等级 | 每分钟频次 | 建议间隔 |
|---------|-----------|---------|
| 0-119 分 | 50 次 | 1.2秒 |
| 120-1999 分 | 120 次 | 0.5秒 |
| 2000+ 分 | 200+ 次 | 0.3秒 |

### 9.3 返回数据格式

**历史K线数据列**（daily）：
```
['ts_code', 'trade_date', 'open', 'high', 'low', 'close', 'pre_close', 'change', 'pct_chg', 'vol', 'amount']
```

**示例数据**（2024-01-02）：
```
ts_code: 600519.SH
open: 1910.00
high: 1915.00
low: 1890.00
close: 1895.00
vol: 27512.85
```

**通用行情接口列**（pro_bar）：
```
['ts_code', 'trade_date', 'open', 'high', 'low', 'close', 'pre_close', 'change', 'pct_chg', 'vol', 'amount']
```

### 9.4 调用示例

```python
import tushare as ts

# 设置 Token（首次使用需要）
ts.set_token('your_token_here')
pro = ts.pro_api()

# 获取股票列表
df = pro.stock_basic(
    exchange='',
    list_status='L',
    fields='ts_code,symbol,name,area,industry,list_date'
)
print(f"获取 {len(df)} 只股票")

# 获取日线行情
df = pro.daily(
    ts_code='600519.SH',
    start_date='20240101',
    end_date='20240131'
)
print(f"获取 {len(df)} 条K线数据")

# 获取前复权行情
df = ts.pro_bar(
    ts_code='600519.SH',
    adj='qfq',
    start_date='20240101',
    end_date='20240131'
)
print(f"获取 {len(df)} 条前复权数据")

# 获取财务指标
df = pro.fina_indicator(
    ts_code='600519.SH',
    period='20231231'
)
print(f"获取 {len(df)} 条财务指标数据")

# 获取龙虎榜
df = pro.top_list(trade_date='20240115')
print(f"获取 {len(df)} 条龙虎榜数据")
```

### 9.5 遇到的问题及解决方案

**问题1：利润表/资产负债表/现金流量表返回0条数据**
- 可能是测试参数（period='20231231'）与实际数据不匹配
- 建议尝试不同报告期参数
- 接口本身可用，需调整查询参数

**问题2：分钟数据需要较高积分**
- stk_mins 接口需要 2000 积分
- 基础积分用户可使用日线数据替代
