## 前端导航结构

对应 `frontend/src/core/router/module_loader.ts` 中的配置：

```

├── 仪表板 (dashboard)                # /dashboard
├── 智能选股 (screener)               # /screener
├── AI 问股 (ask-stock)               # /ask-stock
├── TradingAgents 分析工具
│   ├── 单股分析                      # /trading-agents/analysis/single
```
