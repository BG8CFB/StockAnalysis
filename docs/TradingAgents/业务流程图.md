## 一、文档概述

本文档基于 TradingAgents 模块的实际业务代码，绘制完整的业务流程图，用于业务审查和技术理解。文档涵盖从用户发起任务到生成最终报告的完整流程，包括任务管理、四阶段工作流执行、数据流转和状态管理等核心业务逻辑。

TradingAgents 是基于 LangChain 的多阶段 AI 股票分析系统，模拟专业投资分析团队的工作模式，通过四阶段流程对投资标的进行深度分析。系统采用 Python 原生 async/await + LangChain create_agent 架构，利用 asyncio.gather() 实现真正的并行处理，具备完善的并发控制、任务管理和实时进度推送能力。

本文档的目标读者包括产品经理、业务分析师、开发工程师和测试工程师，旨在帮助相关人员快速理解系统的业务逻辑和数据流转，为功能迭代和问题排查提供参考依据。
